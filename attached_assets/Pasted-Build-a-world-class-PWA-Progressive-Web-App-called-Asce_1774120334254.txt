AI INTEGRATION — Google Gemini API (Free Tier):

Integrate Gemini 1.5 Flash via REST API
API key entered once in Settings, saved to localStorage
Core AI feature: Voice-to-Schedule Assistant

User clicks 🎤 button on dashboard
Web Speech API captures voice → converts to text
Text sent to Gemini with prompt: "Parse this user input and extract: work hours (day + start time + end time), learning goals, studies (courses + days), gym (days + duration), habits. Return as structured JSON."
Gemini understands intent and context, returns JSON
App automatically populates schedule with parsed data
Example: User says "I work Monday to Friday 10 AM to 8 PM, learning C Language and Data Analysis, studies Friday Saturday Sunday"
Gemini extracts and returns: { work: [...], learning: [...], studies: [...] }
Schedule updates instantly ✅


Additional AI features:

Daily briefing: Gemini suggests priorities based on schedule and deadlines
Learning roadmaps: When user adds a learning goal, Gemini generates multi-phase milestone plan
Weekly summary: AI reflects on progress and suggests focus areas


All AI content marked with subtle ✨ indicator
Graceful offline fallback

Auto-Scheduling Engine — Combined Methodologies:

Time Blocking (Primary): Lock work, lock sleep, distribute learning into gaps
Spaced Repetition (Learning): Learning goals spread across multiple days for retention, not cramming
GTD + Eisenhower (Prioritization): High-importance goals get prime time slots, deadlines drive scheduling
Algorithm:

Priority 1 (Locked): Work shifts
Priority 2 (Locked): Sleep (7 hrs default)
Priority 3 (Semi-fixed): Gym, habits
Priority 4 (Flexible): Learning, studies — distributed using spaced repetition across available slots


"🔄 Regenerate" button reshuffles flexible blocks
"✏️ Edit" on any block for manual override
